== Description ==

WP Homepage Scheduler is a plugin for WordPress that allows you to set up a schedule for 
which page will be used as the static frontpage on your WordPress website.

Imagine if you have an event comming up, and you've created an entierly new page that you want to use as 
the main page for your website for a specific timespan, you can then use this plugin to setup the 
timespan you want the specific page to be used for the frontpage, then after that timespan, 
your default frontpage will be used instead.


== Installation ==

1. Download wp-homepage-scheduler.zip to your computer
2. Go to Plugins -> Add New
3. Click the Upload Plugin button.
4. Click the "Select File" dialog and select the wp-schedule-frontpage.zip file you downloaded previously
5. Click Install Now
6. If successful, click Activate Plugin to activate it.
